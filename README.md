# craft-cli

A simple CLI tool for the Huma Craft Cloud platform.

## Installation

### Using Homebrew

```sh
brew tap huma-engineering/craft-cli
brew install craft-cli
```

Or, for a one-liner:

```sh
brew install huma-engineering/craft-cli/craft-cli
```

## Usage

After installation, you can use `craft-cli` directly from your terminal:

```sh
craft-cli --help
```

This will display all available commands and options.
